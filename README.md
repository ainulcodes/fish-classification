# Sistem Klasifikasi Ikan Air Tawar Pemancingan

Aplikasi web untuk mengklasifikasi ikan air tawar menggunakan teknologi CNN (Convolutional Neural Network). Sistem ini dapat mengidentifikasi berbagai jenis ikan air tawar yang populer untuk pemancingan.

## Fitur

- **Klasifikasi Otomatis**: Upload foto ikan dan dapatkan hasil klasifikasi otomatis
- **Database Lengkap**: Informasi lengkap tentang berbagai jenis ikan air tawar
- **Riwayat Klasifikasi**: Simpan dan kelola riwayat hasil klasifikasi
- **User-Friendly Interface**: Antarmuka yang mudah digunakan dengan drag & drop

## Jenis Ikan yang Didukung

- Nila (Oreochromis niloticus)
- Mas (Cyprinus carpio)
- Lele (Clarias batrachus)
- Mujair (Oreochromis mossambicus)
- Gurame (Osphronemus goramy)
- Patin (Pangasius hypophthalmus)
- Bawal Air Tawar (Colossoma macropomum)
- Gabus (Channa striata)
- Dan lainnya

## Tech Stack

### Backend
- FastAPI (Python)
- MySQL 8.0
- SQLAlchemy ORM
- OpenCV & Pillow untuk image processing

### Frontend
- React 19
- Tailwind CSS
- shadcn/ui components
- Axios
